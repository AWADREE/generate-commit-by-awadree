# Support

This is an unofficial extension and is not supported by OpenAI, Microsoft, GitHub, or the VS Code team.

For extension problems, include:

- VS Code version.
- Extension version.
- Operating system.
- Codex CLI version from `codex --version`.
- Whether the relevant changes were staged or unstaged.
- The full error notification text, with secrets removed.

For Codex CLI account, authentication, billing, or service issues, use the official support channels for the Codex CLI or the account provider. This extension does not manage Codex credentials directly.
